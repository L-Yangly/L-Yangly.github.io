摘要：screen是linux下的一种视窗多重复用管理程序。

多视窗 Screen

创建

1

$ screen -s [windowName]

挂起

1

[键盘命令] ctrl+a+d

查看

1
2
3

$ screen -ls
# detach（离线）或者attach（在线）
# 状态为dead，那么用screen -wipe id清理

返回

1

$ screen -r id

离线

1

$ screen -d id

退出

1
2

$ screen -r id
$ exit
